# PromptOps Claude Code plugin

The thin, per-repo artifact from ADR-0009 — no compiled code, just hooks, skills, and an MCP server registration that talk to the PromptOps daemon over `localhost`. See [`../docs/installing-promptops.md`](../docs/installing-promptops.md) for install instructions and [`../docs/plugin-authoring.md`](../docs/plugin-authoring.md) for the hook contract.
